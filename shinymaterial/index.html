shinymaterial
is an R package that implements Google's Material design in Shiny applications via the Materialize library


Getting Started

After installing shinymaterial
install.packages(shinymaterial)
use with the function
material_page(...)
to build the UI

Starter Templates

All starter templates are single file shiny apps (ui and server)
shinymaterial
is only used in ui.R (not server.R)

Basic

library(shiny)
library(shinymaterial)

# Wrap shinymaterial apps in material_page
ui <- material_page(
  title = "Basic Page",
  tags$h1("Page Content")
)

server <- function(input, output) {
  
}
shinyApp(ui = ui, server = server)

Basic + Side-Nav + Tabs

library(shiny)
library(shinymaterial)

# Wrap shinymaterial apps in material_page
ui <- material_page(
  title = "Basic Page + Side-Nav + Tabs",
  # Place side-nav in the beginning of the UI
  material_side_nav(
    fixed = FALSE,
    tags$h3("Side-Nav Content")
  ),
  # Define tabs
  material_tabs(
    tabs = c(
      "First Tab" = "first_tab",
      "Second Tab" = "second_tab"
    )
  ),
  # Define tab content
  material_tab_content(
    tab_id = "first_tab",
    tags$h1("First Tab Content")
  ),
  material_tab_content(
    tab_id = "second_tab",
    tags$h1("Second Tab Content")
  )
)

server <- function(input, output) {
  
}
shinyApp(ui = ui, server = server)

All inputs are accessed via the function
material_input


The following parameters are required:
  • type
  • input_id
  • label
*Some inputs have additional parameters

Use the

type

parameter to utilize a particular input

See

?material_input

for a list of all available types, along with their required parameters

material_input(
  type = "button",
  input_id = "example_button",
  label = "Button",
  icon = "no_icon"
)
